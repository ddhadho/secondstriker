# Second Striker

Second Striker is an application for organizing, managing, and scheduling e-football matches.

## Tech Stack

**Frontend:** Quasar, Vue.js
**Backend:** Node.js, Express.js
**Database:** MongoDB
**Payments:** M-Pesa
